# Personal Website Monorepo

This repository contains the portfolio site and the Sanity Studio used to manage content.

## Projects

- `portfolio-astro/` - Astro frontend site
- `studio-portfolio/` - Sanity Studio project

## Getting Started

### 1) Astro site

```bash
cd portfolio-astro
npm install
npm run dev
```

### 2) Sanity Studio

```bash
cd studio-portfolio
pnpm install
pnpm dev
```

## Notes

- This is a single git repository at the root.
- Environment files are ignored (`.env`, `.env.*`) except examples.
